# 指标监控 Prometheus + Grafana + Exporter

# 简介

# Prometheus

  • 开源的系统监控报警平台
  • 以HTTP协议采集时许数据，并存储在本地数据库
  • 支持服务发现机制，自动完成数据采集
  • 平台本身提供PromQL 查询方式和HTTP接口，以及GUI组件展示数据

# Grafana

  • 开源的可视化监控指标展示平台
  • 支持各种指标图表、仪表盘，并有公共开源仓库
  • 支持多租户、角色等多维度权限控制
  • 支持多种数据库源集成，与Prometheus集成度高
  • 支持警报通知，插件方式很多

# Exporter

  • 采集数据的服务节点
  • 基于Prometheus的服务发现机制，自动采集数据
  • 官网：Grafana (opens new window)、Prometheus (opens new window)
  • 参考：博客 (opens new window)、博客 (opens new window)

# 访问测试

镜像更新升级，配置可能会变

  • 可以先部署容器，拷贝默认配置出来再部署
  • 访问网页，选择Status-Configuration页面，可以看到有效的配置属性

Status-Targets页面，是监控组件的信息，类似Euraka注册中心，可以看到配置中注册的监控节点，节点服务从Prometheus库 (opens new window)中选择，或者自制。
