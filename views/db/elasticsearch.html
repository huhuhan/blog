                Elasticsearch 相关总结

                寒铁 2020-10-10 db

                # 简介

                • 分布式数据库，单个ES实例称为节点，可组成集群
                • 每个数据字符串，分词，处理后写入成一个反向索引，查询数据库，即查询这个反向索引
                • 参考文档 (opens new window)

                # 单位概念

                • Cluster：集群，由一个或多个节点组成
                • Node：单个 ES 实例，即节点
                • Document：ES 是面向文档的，即最小数据单元是文档，通常是数据的JSON表示形式
                • Type：文档的逻辑容器，聚合分组用处，默认**_doc**，6.0后一个Index一个Type，后续版本会删除这概念
                • Index：文档的集合，
                • shared & replica：分片和副本，分布式搜索自动管理，默认1个Index1个分片1个副本（7.x版，之前5个分片）

                关系数据库(MYSQL) Elasticsearch
                数据库(Database) 索引(Index)
                表(Table) 类型(type)
                记录(Record) 文档(Document)
                列(Column) 字段(Field)
                索引(Index) Everything is indexed
                SQL增删改查 REST ful接口

                ​ 关系型数据库，如shecma定义规则， ​ ES，接口参数，用mapping字段定义规则，如索引类型、是否需要分词处理、如何进行分词处理等

                # 倒排索引

                • 正向索引是通过key找value，反向索引则是通过value找key。倒排索引也叫反向索引

                • 一个数据存储ES后

                  • 数据以JSON形式存储，标为一个文档，有文档ID

                  • JSON中的内容数据值，根据定义的分词插件，拆分成单词存储，形式如下

                    • 单词词典：文档中所有单词的集合，不重复。（底层B+树和Hash表类似mysql查询）
                    • 倒排列表：记录单词在哪个文档中出现

                # ELK

                • E：elasticsearch，分布式存储以及全文检索
                • L：logstash，日志加工
                • K：kibana，数据可视化展示

                # 特点优势

                • 分词存储，每个字段被索引，使其可搜索到
                • 分布式搜索，每个Index可拆分多个分片，集群中各节点存储
                • 单个实例到集群再到多台服务器，处理PB级别数据
                • 插件机制，功能扩展，如中文分词插件、同步插件、可视化插件等等

                # 接口

                • postman (opens new window)：测试数据，下载后postman

                • 新增：PUT

                • 搜索：GET，多个索引，用逗号分割

                • 删除：DELETE

                • ES参数，要求Content-Type: application/json

                  • 新增数据：数据JSON格式

                  • 新增索引：配置JSON格式

                    //如配置，默认建的索引，分片为5，副本为1，7.X版本默认1分片1副本
{
    "settings" : {
        //分片
        "index.number_of_shards" : 2,
        //副本
        "index.number_of_replicas" : 1
    }
}
