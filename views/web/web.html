# 查看版本、命令帮助

# 查看现有版本列表
nvm list

# 切换乱码

cmd命令窗口，先输入chcp 65001，在输入切换名， 乱码就正确的。

一般问题是：没有权限！

cmd命令窗口用管理员权限打开操作

# 升级NPM

因为nvm安装后，node目录是快捷映射目录，无法更新里面的npm执行文件。

  1. nvm目录中，删除当前node版本目录下的npm、npx相关文件

  2. 进去当前node版本目录node_modules下，将npm目录先重命名备份，比如xx

  3. 进去该别名的xx目录bin下，用npm-cli.js安装，命令如下

    node npm-cli.js i -g npm
    1

  4. xx目录同级下会生成新的npm目录，xx目录即可删除了

# 初始化，根据提示填写信息
npm init
# 指定官方仓库，项目中可能是taobao镜像仓库，需要切换。发布后记得切换回来不影响本地项目的构建
npm config set registry https://registry.npmjs.org
# 登录官方仓库，根据提示填写信息。需要在官网注册账户
npm adduser
# 发布。发布失败，一般是包名重复，改下package.json的name属性。
npm publish

用此命令构建

cnpm install

# 疑难

# 浮点数运算

# 例子
0.1+0.2
# 结果
0.30000000000000004
1
2
3
4

原理：计算是通过二进制运算的。某些数值转化而二进制是无限长度的。计算机只截取支持的位数（52），再运算，最后转化回十进制的值。

0.1 => 0.0001 1001 1001 1001…（无限循环）
0.2 => 0.0011 0011 0011 0011…（无限循环）
1
2

方案：

  • 先乘再除：保证计算需要的结果值是整数，再除掉。最后用toFixed(保留数)来截取长度
